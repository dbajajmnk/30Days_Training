# Bruno Collection (v3) – Enterprise Frontend Module BFF

## Import (Bruno v3+)
1. Extract the zip
2. Bruno → **Open Collection**
3. Select the extracted folder (the one containing `bruno.json`)

## Use
1. Select environment: `local`
2. Run `Auth Login (ADMIN)` (or STAFF/CLIENT)
3. Copy `accessToken` from response → paste into env var `accessToken`
4. Run: Auth Me, Users List, Dashboard, Users Create (ADMIN)
